### input format

```
v \in V = {1, 2, 3, ... , n}
p_i is the position of a vertex
n = | V |
m = | E |
k is the num of page
k = 1, 2, ..., n


===input sample===
n m
k
v_1 u_1
v_2 u_2
v_3 u_3
...
v_m u_m

===output sample1===
n m
k
p_1
p_2
...
p_n
v_1 u_1 [j]
v_2 u_2 [j]
...
v_m u_m [j]

===output sample2(case one page)===
n m
k
p_1
p_2
...
p_n
v_1 u_1
v_2 u_2
...
v_m u_m
```
